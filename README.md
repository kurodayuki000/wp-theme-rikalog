## サーバーにアップロードが必要なファイルは以下です。                                                                                                                             
                                                                                                                                                                               
アップロードするの                                                                                                                                                           
                                                                                                                                                                            ```
rikalog-theme/                                                                                                                                                                 
  ├── style.css                  ← WPテーマ識別用（必須）                                                                                                                      
  ├── screenshot.png             ← テーマ画像
  ├── functions.php              ← 変更済み
  ├── header.php
  ├── footer.php
  ├── front-page.php
  ├── single.php
  ├── page.php
  ├── page-about.php
  ├── page-contact.php
  ├── page-blog.php
  ├── archive.php
  ├── category.php
  ├── search.php
  ├── 404.php
  ├── index.php
  ├── sidebar.php
  ├── comments.php
  ├── template-parts/
  │     ├── content.php
  │     ├── content-none.php
  │     └── toc.php
  ├── inc/
  │     └── blocks/
  │           ├── link-card.php  ← パス変更済み
  │           └── marker.php     ← パス変更済み
  ├── assets/                    ← favicon等あれば
  └── dist/                      ← ビルド出力（必須）
        ├── theme.css
        ├── theme.js
        └── blocks/
              ├── link-card.js
              └── marker.js

アップロード不要なもの
┌────────────────────┬─────────────────────────────────────────────────┐
│ ファイル/フォルダ  │                      理由                       │
├────────────────────┼─────────────────────────────────────────────────┤
│ node_modules/      │ サーバーでは不要（開発用）                      │
├────────────────────┼─────────────────────────────────────────────────┤
│ src/               │ ソースコード（ビルド済みの dist/ があれば不要） │
├────────────────────┼─────────────────────────────────────────────────┤
│ package.json       │ 開発用                                          │
├────────────────────┼─────────────────────────────────────────────────┤
│ tsconfig.json      │ 開発用                                          │
├────────────────────┼─────────────────────────────────────────────────┤
│ tailwind.config.ts │ 開発用                                          │
├────────────────────┼─────────────────────────────────────────────────┤
│ esbuild.config.ts  │ 開発用                                          │
├────────────────────┼─────────────────────────────────────────────────┤
│ .gitignore         │ 開発用                                          │
├────────────────────┼─────────────────────────────────────────────────┤
│ js/ (旧フォルダ)   │ dist/ に置き換え済み                            │
└────────────────────┴─────────────────────────────────────────────────┘
```
手順

1. ローカルで npm run build を実行
2. 上記のファイル一式をサーバーにアップロード
3. dist/ フォルダを忘れずに含めること

ポイント: dist/ がないとCSS・JSが読み込まれずサイトが表示崩れします。デプロイ前に必ず npm run build を実行してください。
